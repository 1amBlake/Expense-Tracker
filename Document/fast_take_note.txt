QTNV
- Đăng Ký Tài Khoản
Nhập thông tin
→ Validate
→ Kiểm tra trùng email/username
→ Mã hóa mật khẩu
→ Tạo tài khoản
→ Thành công

- Đăng Nhập
Nhập username/email + password
→ Kiểm tra tồn tại
→ Verify password
→ Tạo session
→ Vào hệ thống

- Tạo Khoản Chi Tiêu
Nhập chi tiêu
→ Validate dữ liệu
→ Kiểm tra nguồn tiền
→ Kiểm tra số dư
→ Kiểm tra ngân sách
→ Lưu expense
→ Trừ số dư nguồn tiền
→ Cập nhật ngân sách
→ Cập nhật thống kê
→ Lưu lịch sử

- Tạo Khoản Thu Nhập
Nhập thu nhập
→ Validate
→ Kiểm tra nguồn tiền
→ Lưu income
→ Cộng số dư nguồn tiền
→ Cập nhật thống kê
→ Lưu lịch sử

- Chuyển Tiền Nội Bộ
Chọn nguồn gửi
→ Chọn nguồn nhận
→ Validate
→ Kiểm tra số dư
→ Trừ nguồn gửi
→ Cộng nguồn nhận
→ Lưu transfer
→ Lưu lịch sử

- Tạo/Quản Lý Ngân Sách
Tạo ngân sách
→ Chọn chu kỳ
→ Nhập giới hạn
→ Validate
→ Lưu ngân sách

- Kiểm Tra Ngân Sách Khi Chi Tiêu
Tạo expense
→ Có ngân sách?
    → Không → lưu
    → Có
        → Tính usage
        → Vượt ngưỡng?
             → Cảnh báo
        → Vượt ngân sách?
             → Đánh dấu over-budget
→ Lưu

- Giao Dịch Định Kỳ
Scheduler chạy
→ Kiểm tra recurring đến hạn
→ Active?
→ Auto hay manual?
    → Manual → gửi thông báo
    → Auto
         → Kiểm tra số dư
         → Tạo transaction
         → Cập nhật balance
         → Lưu history
		 
- Soft Delete
Người dùng xóa transaction
→ Chuyển deleted = true
→ Cập nhật balance
→ Cập nhật budget
→ Chuyển recycle bin

- Restore
Restore transaction
→ deleted = false
→ Recalculate balance
→ Recalculate budget
→ Restore history

- Sao Lưu Dữ Liệu
Chọn backup
→ Validate dữ liệu
→ Export DB/file
→ Thông báo thành công

- Khôi Phục Dữ Liệu
Chọn file backup
→ Validate format
→ Kiểm tra version
→ Confirm overwrite
→ Restore data
→ Reload hệ thống

- Xem Báo Cáo Thống Kê
Chọn khoảng thời gian
→ Query dữ liệu
→ Tính toán
→ Group dữ liệu
→ Render chart/statistics
